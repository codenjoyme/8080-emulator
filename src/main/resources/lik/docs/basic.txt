Основные отличия команд редактирования языка "Basic Лик" от других:

EDIT   -        редатктирование строки
CONT   -        продолжить выполнение программы после ее остановки
AUTO   -        включение автонумерации строк. Выход из режима клавиша "[/]"
                (перечеркнутый квадрат)
MSAVE  -        вывод на магнитофон. Имя должно состоять из одного символа.
(имя)           Если имя не указано то впоследствии будет загрузка программы
                без имени.
MLOAD  -        ввод с магнитофона. Если указано имя, то загружается программа
(имя)           с именем. Если нет - первая попавшаяся программа, без имени.
NEW    -        обязателен при первом запуске интерпретатора
RUN (n)-        запуск со строки номер n, или просто запуск программы.
LIST (n)-       распечатка со строки n, или распечатка всей программы.
                Приостановить распечатку - <STOP>, продолжить <ВК>.
                Прекратить распечатку - [/] (перечеркнутый квадрат)

 Во время ввода строки можно вводить операторы и имена функций последовате-
льным нажатием двух клавиш: Home и символьной клавиши.
