 1.3. Память

Радиоконструктор ЛИК мажет работать как с памятью только для чтения (ПЗУ).
так и с памятью для чтения/записи (ОЗУ). Программа может читать данные из па-
мяти любого типа,а записывать только в ОЗУ.

Для программиста память представляет собой последовательность байтов , где
могут находиться данные или команды. Адреса байтов памяти могут быть от О до
65535=FFFFH.

В радиоконструкторе ЛИК память распределена следующим образом:

ОЗУ:            0000H - 80FFH - рабочая область для программиста;
                0000H - 1DFFH - область загрузки "BASIC";
                0000H - 0FFFH - область загрузки "РЕДАКТОРА";
                1100H - 20FFH - область трансляции;
                2100Н         - начало текстового буфера;
                8100H - 8B3FH - область загрузки "МОНИТОР-1M" с ленты;
                8B40H - 8F3FH - область каталога ленты;
                9000H - BFFFH - область экранного ОЗУ.
ПЗУ:            C000H - C7FFH - "ЗАГРУЗЧИК";
                C800H - D23FH - "МОНИТОР-1М";
                D240H - EFFFH - "BASIC".

Рабочие ячейки ОЗУ:
8FFFH - константа ввода (начальное значение ЗСН) ;
8FFEH - константа вывода (начальное значение 28Н);
8FFAH,8FFBH - код цвета экрана после очистки:
            00 - черный;
            FF - белый;
8FFCH ,8FFDH - координаты курсора (вертикальная и горизонтальная).
