Людвиг Ван Бетховен
"Соната No.14"


Описание

Демонстрация звуковых возможностей ПК "Специалист".

Программа проигрывает одну из самых известных и к слову одну из самых любимых мной, санат Людвига Ван Бетховена - "Лунную Санату". Конечно воспроизведение далеко от оригинала, но узнаваемо :) Для вывода звука используется алгоритм 2-х голосой эмуляции звука. Сей алгоритм был использован во многих играх для ПК "Специалист", да и я грешным делом, по молодости, использовал сам код подобного вывода звука для своей программы "Музыкальная шкатулка". Вот только найти бы ее еще на катушках... Правда код я рипнул не из данной программы, а из игрушки "Стена".
